🎓 Project Overview

School Attendance & Performance Management System
This is a web-based system built for teachers to:
Manage students
Track attendance
Record marks
Predict performance
Analyze student data

🧱 Core Technologies Used

Frontend:
HTML
CSS
JavaScript (ES Modules)
Backend:
Firebase Authentication → login system
Firebase Realtime Database (RTDB) → data storage
Libraries:
Chart.js → graphs & analytics
